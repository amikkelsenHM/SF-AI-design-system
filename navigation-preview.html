Sidenav Component Preview

Interactive preview of the Spaceflux sidenav component with toggle functionality.

Component Features

This sidenav component includes the following features:

  • Two States: Wide (252px) and Collapsed (72px)
  • Smooth Transitions: Animated state changes
  • Active State Indicator: Purple left border and darker background
  • Hover Effects: Visual feedback on interaction
  • Responsive Logo: Full wordmark in wide state, icon in collapsed state
  • Toggle Button: Click the circular button to expand/collapse

Design Specifications

Colors:

  • Background: #100919 (Surface Dark)
  • Active Background: #36274A (OnDark Contrast)
  • Active Border: #CF8BFF (Medium Orchid)
  • Text: #EEEEEE (OnDark)

Typography:

  • Font: IBM Plex Sans Bold
  • Size: 12px
  • Line Height: 12px

Spacing:

  • Item Padding: 24px
  • Icon-Label Gap: 12px
  • Icon Size: 24x24px
  • Active Border Width: 4px

Usage Example

To use this component in your prototype:

<nav class="sidenav sidenav-wide"> <div class="sidenav-logo"> <img src="logo.svg" alt="Logo" class="sidenav-logo-full"> <img src="logo-icon.svg" alt="Logo" class="sidenav-logo-icon"> </div> <div class="sidenav-nav"> <a href="#page1" class="sidenav-item sidenav-item-active"> <div class="sidenav-icon">[icon]</div> <span class="sidenav-label">Page Name</span> </a> </div> </nav>

Customization

You can easily customize the navigation items by:

  • Adding or removing .sidenav-item elements
  • Changing the href attributes to link to your pages
  • Updating icon SVGs to match your needs
  • Modifying labels to match your prototype sections
  • Adding .sidenav-item-active class to indicate current page

Try It Out

Click the circular toggle button on the left side of the navigation to see it collapse and expand. Notice how:

  • The logo switches between full wordmark and icon
  • Labels fade out smoothly in collapsed state
  • The main content area adjusts automatically
  • All transitions are smooth and polished
